# Dataset Documentation Template

This template is used to document the datasets stored in this repository (e.g., inside `datasets/demo/` or `datasets/uploaded/`).

## Template Fields

### Dataset Name
*Name or ID of the dataset (e.g., LISS-IV scene identifier)*

### Source
*Origin/source of the dataset (e.g., ISRO, Bhuvan portal, NRSC)*

### Acquisition Date
*Date and time when the satellite image was captured (e.g., YYYY-MM-DD)*

### Location
*Geographic area covered by the dataset (e.g., Region name, bounding box coordinates)*

### Coordinate System
*Coordinate reference system (e.g., WGS 84 / UTM Zone 43N, EPSG:32643)*

### Bands Available
*Spectral bands included in the dataset (e.g., Band 2 (Green), Band 3 (Red), Band 4 (Near Infrared))*

### Resolution
*Spatial resolution of the imagery (e.g., 5.8 meters)*

### Purpose
*The intended use or research objective of this specific dataset*

### Notes
*Any additional metadata, cloud cover percentage, atmospheric correction status, or general notes*
